Stratégie des tokens

🎟️

Access Token

TTL 15 minutes — Stocké en mémoire. Envoyé dans chaque requête API via header Authorization.

🔄

Refresh Token

TTL 1 jour — Hashé en base (RefreshToken collection). Révocable à tout moment (logout, sécurité).

🛡️

Interceptor Angular

Intercepte les erreurs 420 — Renouvelle le token automatiquement et rejoue la requête. Invisible pour l'utilisateur.
